日志查询常用语句

查询一段时间内接口调用方和调用量

requests
| where operation_Name  ==  "接口"  and timestamp >=ago(时间)
| extend p_url= parse_url(url)
| extend platform=  p_url["Query Parameters"]["Platform"]
| summarize c= count(platform) by tostring(platform)
| project  c,platform

其中platform 是属于url后面的查询参数 eg: www.baidu.com?platform=1123 是我们公司用来统计调用平台的,可根据实际情况 自行替换
[“Query Parameters”] 可以获取url里的查询参数

查询一段时间内依赖项响应情况

dependencies
| where timestamp >=ago(7d) and name contains "依赖项接口名称" 
| summarize avg(duration) by bin(timestamp, 10m) 
| render timechart

一段时间内各接口速度和调用数量

requests
| extend OperationName = replace(@"=.|\d{6,}|@=/.\d+.@<=/|Code/.\d+.*",@"{id}", tostring(operation_Name))
| summarize AvgDuration = avg(duration),Count = count() by OperationName
| order by Count desc
| project AvgDuration, OperationName, Count

最近一天的sql平均响应时间和数量

dependencies
| where timestamp > bin(ago(2d), 1d) and timestamp < bin(now(), 1d) and type == 'SQL'
| summarize Count = count(data), AvgDuration = avg(duration) by data
| order by Count desc
| project Count, AvgDuration, data
